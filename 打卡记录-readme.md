## 🚫 **Project Mint - 反向 README**

打卡记录器。

### ❌ **不能干什么**

**替代方案**：可以找任何一个 AI 改代码。

#### 🎨 **不支持自定义主题**
- 薄荷绿（`#A3E4D7`），换了颜色就不是“薄荷”了。

#### 📳 **不支持取消振动反馈**

- 个人爱好。

#### 🏷️ **不支持多选类别**
- 一个习惯只能属于一个分类。它可以是运动**或**工作，不能是运动**和**工作。

#### 🔔 **不支持定时提醒**
- 不会主动弹窗说“该打卡了”。

#### 🏆 **不支持成就系统**
- 没有“连续打卡7天”、“累计100次”之类的徽章弹窗。

#### 📊 **不支持数据可视化（热力图/折线图）**
- 没有GitHub风格的绿色格子/“过去30天打卡趋势图”。

#### 🌐 **不支持云端同步**
- 所有数据存在手机本地，不会上传任何服务器。

#### 👥 **不支持多用户**
- 没有“多人共享”、“团队打卡”功能。

#### 🧩 **不支持“子步骤嵌套子步骤”**
- 每个习惯下的步骤只有一层，不能再拆分子步骤。
- **原因**：再拆下去就变成“项目管理”了，不是“习惯打卡”。

#### 📤 **不支持“分享到社交平台”**
- 没有“一键分享”。

#### 🔄 **不支持“自动补签”**
- 与其纠结昨天，不如专注今天。

#### **❌ 不支持“AI分析/自动规划习惯”**

- 只有你自己知道什么值得坚持。
- 接入 AI 花钱。

---

### ✅ **但它能做到这些事**

| 功能                              | 状态   |
| --------------------------------- | ------ |
| Android/iOS/web端             | ✅ 支持 |
| 深色/浅色模式跟随系统             | ✅ 支持 |
| 搜索习惯                          | ✅ 支持 |
| 分类筛选                          | ✅ 支持 |
| 多选批量操作（打卡/删除）         | ✅ 支持 |
| 侧滑手势拦截（多选时）            | ✅ 支持 |
| 子步骤快捷触发模式（按一下+1）    | ✅ 支持 |
| 子步骤连招模式（全选完成+1）      | ✅ 支持 |
| 自动保存                          | ✅ 支持 |
| 长按卡片呼出详情面板              | ✅ 支持 |
| 详情面板内修改分类/名称/备注/步骤 | ✅ 支持 |

#### 🧩 **微型连招**

- **子步骤管理**：可将一个习惯拆解为多个微型步骤，支持任意增删改。
- **两种触发模式**：
  - **连招模式 (AND)**：必须全部步骤完成，主项目才 +1，适合固定流程（如“早晨唤醒”）。连招模式下，全部步骤完成后自动增加打卡次数并重置步骤。
  - **⚡ 快捷触发模式 (OR)**：点击任意子步骤直接为主项目 +1，适合碎片化高频打卡（如“运动”中的深蹲、俯卧撑）。

**这个软件不能做的事非常多。但它能做的事，每一件都做得很好。**

它不是万能的，但它恰好是我需要的。 😏🐍
